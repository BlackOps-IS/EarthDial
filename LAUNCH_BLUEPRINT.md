## Component Detail

### Forecast Ingestion Layer
- **Input Formats:** NetCDF4, GRIB2, JSON (Earth-2 compatible)
- **Processing:** Extract wind vectors, temperature grids, humidity fields, precipitation forecasts
- **Caching:** Redis-backed forecast cache with 15-minute TTL
- **Earth-2 Bridge:** Direct ingestion of FourCastNet/CorrDiff ensemble outputs
- **Latency Target:** <100ms per forecast frame ingestion

### Risk Computation Engine
- **Model:** Rothermel-inspired ignition probability with compound interaction terms
- **Grid:** 40Ã—40 (1,600 cells, ~550m resolution), extensible to arbitrary resolution
- **Acceleration:** NumPy vectorized (current) â†’ cuDF/cuPy (production path)
- **Fire Spread:** Elliptical cone model with wind-direction elongation
- **Latency Target:** <50ms for full grid risk computation

### Graph Optimization Engine
- **Model:** Power grid as NetworkX graph (substations=nodes, lines=edges)
- **Algorithm:** Exhaustive combinatorial search with constraint satisfaction
  - Objective: maximize risk_reduction / disruption_score
  - Constraints: grid connectivity, critical load preservation
- **Acceleration:** NetworkX (current) â†’ NVIDIA cuGraph (production path)
- **Latency Target:** <200ms for 8-line grid, <2s for 100-line grid (cuGraph)

### Nemotron Reasoning Engine
- **Model:** `nvidia/llama-3.3-nemotron-super-49b-v1`
- **API:** `integrate.api.nvidia.com/v1/chat/completions`
- **Capabilities:**
  - Prevention brief generation (structured 9-section document)
  - Counterfactual causal explanation
  - Community alert synthesis
- **Deployment Path:** NVIDIA NIM container â†’ Triton Inference Server
- **Latency Target:** <3s for prevention brief, <2s for counterfactual explanation

### Multi-Agent Reasoning Orchestration (Roadmap)
- **Risk Assessment Agent:** Continuous monitoring, threshold alerting
- **Optimization Agent:** Autonomous shutoff planning, constraint evaluation
- **Communication Agent:** Stakeholder notification, document generation
- **Coordination Agent:** Multi-agent consensus, conflict resolution
